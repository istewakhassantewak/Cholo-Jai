# Cholo Jai
Cholo Jai is a modern and responsive travel landing page built using HTML and CSS. The project focuses on delivering a clean user interface, engaging layout, and smooth responsiveness across all devices.

🚀 Live Demo
🔗 https://cholo-jai-six.vercel.app/

📌 Features
Fully responsive design (Mobile, Tablet, Desktop)
Clean and modern travel UI
“Choose Destination” section
Structured layout with proper spacing & alignment
Lightweight and fast performance
Simple and user-friendly navigation
🛠️ Technologies Used
HTML5
CSS3

⚙️ Setup & Usage
Clone the repository:
git clone https://github.com/istewakhassantewak/cholo-jai.git
Navigate to the folder:
cd cholo-jai
Open index.html in your browser.

No frameworks or build tools required — pure HTML & CSS.
